# 🚗 Car Game

A fun and interactive car racing game where you avoid traffic and score points! This is a Progressive Web App (PWA) that can be installed on any device and played offline.

## 🎮 Features

✅ **Installable PWA** - Install on any device (mobile, tablet, desktop)  
✅ **Offline Support** - Play without internet connection  
✅ **Progressive Difficulty** - Levels increase with your score  
✅ **Best Score Tracking** - Your high scores are saved locally  
✅ **Pause/Resume** - Full game controls  
✅ **Mobile Responsive** - Works on all screen sizes  
✅ **Beautiful UI** - Modern dark theme with smooth animations  

## 🎯 How to Play

1. Click **"Start Game"** to begin
2. Use **Arrow Keys** or **WASD** to move your car
3. **Avoid the red traffic cars** coming down the road
4. **Don't hit the walls** or yourself
5. Each successful avoidance = **+10 points**
6. Survive longer to reach higher levels and increase difficulty

## 📱 Installation

### On Desktop:
1. Visit the game URL
2. Look for the **"Install App"** button or browser install prompt
3. Click to install

### On Mobile:
1. Visit the game URL in your browser
2. Tap the menu (⋮ or ⋯)
3. Select "Install app" or "Add to home screen"
4. Confirm installation

## 🎮 Controls

| Key | Action |
|-----|--------|
| ⬅️ Arrow Left / A | Move Left |
| ➡️ Arrow Right / D | Move Right |
| ⬆️ Arrow Up / W | Move Up |
| ⬇️ Arrow Down / S | Move Down |
| Space | Pause/Resume |

## 📊 Scoring

- **+10 points** per traffic car avoided
- **Level increases** every 100 points
- **Speed increases** as difficulty goes up
- **Best score saved** automatically

## 🛠️ Technical Details

- Built with vanilla **HTML5**, **CSS3**, and **JavaScript**
- Uses **Canvas API** for rendering
- Progressive Web App with **Service Worker** for offline support
- Responsive design using **CSS Media Queries**
- Local storage for high scores

## 📦 Files

- `index.html` - Main game interface
- `game.js` - Game logic and mechanics
- `manifest.json` - PWA configuration
- `service-worker.js` - Offline support
- `service-worker-register.js` - Service worker registration

## 🚀 Deployment

This game is optimized for GitHub Pages. Simply:

1. Create a GitHub repository named `car-game`
2. Push all files to the repository
3. Enable GitHub Pages in repository settings
4. Your game will be live at: `https://yourusername.github.io/car-game/`
